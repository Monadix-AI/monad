---
name: summarize-changes
description: Summarize uncommitted git changes and flag anything risky. Use when the user asks what changed, wants a commit message, or asks to review their diff.
---

Summarize the current uncommitted changes in two or three bullet points, then list any risks
you notice — missing error handling, hardcoded values, secrets, or tests that need updating.

If there are no uncommitted changes, say so plainly.
